# Chang Cookbook Logo Assets

Place your logo files in this directory with these naming conventions:

## Primary Logo Files
- `chang-logo.svg` - Vector format (preferred for web)
- `chang-logo.png` - High resolution PNG (512x512px)
- `chang-logo-small.png` - Small version (96x96px)
- `chang-logo-favicon.png` - Favicon version (32x32px)

## Logo Specifications

### Recommended Dimensions
- **SVG**: Vector format - scalable to any size
- **PNG High-res**: 512x512px (for crisp scaling)
- **PNG Small**: 96x96px (for optimal loading)
- **Favicon**: 32x32px, 16x16px

### Usage Context
- Header: 48px desktop, 40px mobile
- Footer: 32px
- Favicon: 16px, 32px
- Social sharing: 400x400px

### File Format Recommendations
1. **SVG** - Best for web, crisp at any size
2. **PNG** - Good fallback, supports transparency
3. **WebP** - Modern format, smaller file size

### Logo Design Guidelines
- Ensure logo works on both light and dark backgrounds
- Keep adequate contrast with Chang brand colors
- Design should be recognizable at small sizes (16px)
- Maintain circular or square aspect ratio for consistency
